# Data Visualisation Dashboard — Power BI

A Power BI dashboard built as part of university coursework, exploring data visualisation principles and interactive reporting.

This was my first serious Power BI project — the Goomti Restaurant dashboard in my main portfolio is a more advanced version of the skills developed here, applied to real business data.

---

## What's in here

- `Data Visualisation Dashboard.pbix` — the Power BI file (requires Power BI Desktop to open, free download from Microsoft)

---

## Note

If you don't have Power BI Desktop, the Goomti Analytics Portfolio repo includes an HTML version of a more complete dashboard that opens directly in any browser — no software needed.
